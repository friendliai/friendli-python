# DedicatedImageGenerationBodyResponseFormat

The format in which the generated image(s) will be returned. One of `url(default)`, `raw`, `png`, `jpeg`, and `jpg`.


## Values

| Name   | Value  |
| ------ | ------ |
| `URL`  | url    |
| `RAW`  | raw    |
| `PNG`  | png    |
| `JPEG` | jpeg   |
| `JPG`  | jpg    |
