# Serverless.Audio

## Overview

### Available Operations

* [transcribe](#transcribe) - Audio transcriptions

## transcribe

Given an audio file, the model transcribes it into text.

### Example Usage

<!-- UsageSnippet language="python" operationID="serverlessAudioTranscriptions" method="post" path="/serverless/v1/audio/transcriptions" example="Example" -->
```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.serverless.audio.transcribe(
        model="openai/whisper-large-v3",
        file={
            "file_name": "example.file",
            "content": open("example.file", "rb"),
        },
    )

    # Handle response
    print(res)
```

## stream

Given an audio file, the model transcribes it into text.

### Example Usage

<!-- UsageSnippet language="python" operationID="serverlessAudioTranscriptionsStream" method="post" path="/serverless/v1/audio/transcriptions#stream" example="Example" -->
```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.serverless.audio.stream(
        model="openai/whisper-large-v3",
        file={
            "file_name": "example.file",
            "content": open("example.file", "rb"),
        },
        stream=True,
    )

    with res as event_stream:
        for event in event_stream:
            # handle event
            print(event, flush=True)
```
