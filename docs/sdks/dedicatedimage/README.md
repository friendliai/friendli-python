# Dedicated.Image

## Overview

### Available Operations

* [generate](#generate) - Image generations
* [edit](#edit) - Image edits

## generate

Given a description, the model generates image(s).

### Example Usage

<!-- UsageSnippet language="python" operationID="dedicatedImagesGenerate" method="post" path="/dedicated/v1/images/generations" -->
```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.dedicated.image.generate(
        model="(endpoint-id)",
        prompt="An orange Lamborghini driving down a hill road at night with a beautiful ocean view in the background.",
        num_inference_steps=10,
        control_images=[
            {
                "image": "https://huggingface.co/datasets/huggingface/documentation-images/resolve/main/diffusers/cat.png",
            },
        ],
    )

    # Handle response
    print(res)
```

## edit

Given an image and a description, the model edits the image.

### Example Usage

<!-- UsageSnippet language="python" operationID="dedicatedImagesEdit" method="post" path="/dedicated/v1/images/edits" -->
```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.dedicated.image.edit(
        image={
            "image": "https://huggingface.co/datasets/huggingface/documentation-images/resolve/main/diffusers/cat.png",
        },
        prompt="Add a red sports car in the foreground.",
        model="(endpoint-id)",
        num_inference_steps=20,
        guidance_scale=0,
    )

    # Handle response
    print(res)
```
