# Dedicated.Messages

## Overview

### Available Operations

* [messages](#messages) - Messages

## messages

Generate responses using Anthropic Messages-style payloads.

### Example Usage: Example

<!-- UsageSnippet language="python" operationID="dedicatedMessages" method="post" path="/dedicated/v1/messages" example="Example" -->
```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.dedicated.messages.messages(
        messages=[
            {
                "role": "user",
                "content": "Hello, summarize what you can do in one sentence.",
            },
        ],
        model="(endpoint-id)",
        max_tokens=128,
        additional_properties={},
    )

    # Handle response
    print(res)
```
