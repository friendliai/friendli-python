# DedicatedChat
(*dedicated.chat*)

## Overview

### Available Operations

* [complete](#complete) - Chat completions
* [stream](#stream) - Stream chat completions

## complete

Given a list of messages forming a conversation, the model generates a response.

### Example Usage

<!-- UsageSnippet language="python" operationID="dedicatedChatComplete" method="post" path="/dedicated/v1/chat/completions" -->
```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.dedicated.chat.complete(
        model="(endpoint-id)",
        messages=[
            {
                "role": "system",
                "content": "You are a helpful assistant.",
            },
            {
                "role": "user",
                "content": "Hello!",
            },
        ],
        max_tokens=200,
    )

    # Handle response
    print(res)
```

## stream

Given a list of messages forming a conversation, the model generates a response.

### Example Usage

<!-- UsageSnippet language="python" operationID="dedicatedChatStream" method="post" path="/dedicated/v1/chat/completions#stream" -->
```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.dedicated.chat.stream(
        model="(endpoint-id)",
        messages=[
            {
                "role": "system",
                "content": "You are a helpful assistant.",
            },
            {
                "role": "user",
                "content": "Hello!",
            },
        ],
        max_tokens=200,
    )

    with res as event_stream:
        for event in event_stream:
            # handle event
            print(event, flush=True)
```
