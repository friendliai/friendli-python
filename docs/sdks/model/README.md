## Overview

## list

Retrieve list of serverless models.

### Example Usage

```python
import os

from friendli import SyncFriendli

with SyncFriendli(
    token=os.getenv("FRIENDLI_TOKEN", ""),
) as friendli:
    res = friendli.serverless.model.list()

    # Handle response
    print(res)
```
